# hello-vweb

A simple hello world using V &amp; vweb.

To run use: `v run hello-vweb.v`.

Then try:

`curl http://localhost:8080` to see:

```html
<html>
  <header>
    <title>Vweb Hello World</title>
  </header>
  <body>
    <main>Hello World</main>
  </body>
</html>
```

or

`curl http://localhost:8080/v` to see:

```html
<html>
  <header>
    <title>Vweb Hello World</title>
  </header>
  <body>
    <main>Hello v</main>
  </body>
</html>
```
